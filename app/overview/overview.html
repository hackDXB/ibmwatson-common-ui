Overview

The Watson Common UI project provide CSS styles and angular directives for use in Watson Tooling projects.

Some of the directives are provided by Angular UI with styling provided by this project. Do not use the bootstrap CSS.

Using the common styles

The common styles are built into the ibmwatson.css file, so you will need to include this in your index.html. Add the "watson" class to the body element, and the styles will be available.

The CSS is designed to apply to the semantic HTML elements as far as possible, so if you use the correct HTML elements you will automatically pick up the styles.
