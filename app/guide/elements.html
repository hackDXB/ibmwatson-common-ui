Typography

Use the right sematic tags or classes for text

  • Normal paragraph - <p>

  • strong text - <strong>

  • emphasized text - <em>

  • Small text - <small>

  • Helper text - <small class="helper">

  • Uppercase text - <* class="uppercase">

  • Hyperlink - <a>

  • deleted text - <del>

  • highlighted text - <mark>

  • Error text - <* class="error-text">

  • Use <code> for code snippets

Links

Use <a> for links. gray and purple classes available for overriding the color.

Forms and Inputs

Use <form> for forms.

Use the appropriate types for inputs (text, email, url, search, username, password, checkbox, radio). Remember to include a <label for="id"> for inputs (or aria-label if no visible label needed).

Radio Button

Use <input type="radio"> for radio buttons. Remember to wrap the radio group in a <fieldset>.

Checkboxes

Use <input type="checkbox"> for checkboxes.

Use <input type="search"> for search fields.

Buttons

Use <button> for primary buttons. Add a class of secondary, error or link for the others.

Use <button disabled> for disabled buttons. Use ng-disabled to dynamically set this.

Use <button class="large"> for large buttons.

Use <button class="small"> for small buttons.

File upload

Use <input type="file"> wrapped in a <div class="file-upload"> for file inputs.

Tables

Use <table>, <thead>, <tbody>, <tr>, <th> and <td label="..."> for tables.

Progress bars

Use <progress> for progress bars. You can use something like ng-class="{'good': percentageComplete > 50}" to adjust the style as it progresses.
